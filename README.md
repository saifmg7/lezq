# 🔴 منصة هندسة الليزر والإلكترونيات البصرية
## Laser & Photonics Engineering Platform

منصة تعليمية متقدمة واحترافية لدراسة مادة الانتشار في هندسة الليزر والإلكترونيات البصرية.

---

## ✨ المميزات الرئيسية

### 🎯 5 تصورات ثلاثية الأبعاد تفاعلية

1. **📡 انتشار الموجة الضوئية**
   - محاكاة حية لانتشار الموجات الكهرومغناطيسية
   - عرض المجالات الكهربائية والمغناطيسية
   - تفاعل ديناميكي مع الوقت

2. **🔴 شعاع الليزر ثلاثي الأبعاد**
   - تمثيل واقعي لشعاع الليزر المتماسك
   - تأثيرات توهج وإضاءة متقدمة
   - نظام جزيئات ديناميكي

3. **🔬 الألياف البصرية**
   - محاكاة انتقال الضوء عبر الألياف
   - الانعكاس الداخلي الكلي
   - مسارات ضوئية معقدة

4. **🌊 نمط التداخل الضوئي**
   - تداخل موجتين ضوئيتين
   - أنماط مضيئة ومظلمة ديناميكية
   - ألوان متغيرة بناءً على التداخل

5. **🔭 العناصر البصرية**
   - عدسات وموشورات وسطوح عاكسة
   - انكسار وانعكاس الضوء
   - حركة وتفاعل ثلاثي الأبعاد

---

## 🛠️ التكنولوجيات المستخدمة

- **HTML5**: هيكل الصفحة المتقدم
- **CSS3**: تصاميم ديناميكية واستجابة لحجم الشاشة
- **JavaScript ES6+**: منطق تفاعلي متقدم
- **Three.js**: محرك الرسومات ثلاثية الأبعاد

---

## 📋 محتويات الملفات

### `index.html`
- الصفحة الرئيسية المحترفة
- أقسام متعددة (نظرية، تصورات، تطبيقات، تواصل)
- نموذج الاتصال التفاعلي

### `styles.css`
- تصاميم احترافية وحديثة
- تأثيرات hover و animations
- دعم كامل للأجهزة المختلفة (Responsive Design)
- ألوان متناسقة وجذابة

### `script.js`
- وظائف تفاعلية متقدمة
- نظام الإشعارات
- معالجة نماذج الاتصال
- تحسينات الأداء

### `3d-visualizations.js`
- 5 تصورات ثلاثية الأبعاد باستخدام Three.js
- محاكاة الموجات والشعاع والألياف
- أنماط التداخل الديناميكية
- المشاهد والكاميرات والإضاءة

## 🔧 التخصيص

### تغيير الألوان
عدّل متغيرات CSS في `styles.css`:
```css
:root {
    --primary-color: #e74c3c;
    --secondary-color: #3498db;
    --dark-bg: #1a1a2e;
    --light-text: #ecf0f1;
    --accent-color: #f39c12;
    --success-color: #2ecc71;
}
```

### تخصيص التصورات ثلاثية الأبعاد
اعدّل معاملات Three.js في `3d-visualizations.js`:
- `camera.position`: موضع الكاميرا
- `Light intensity`: شدة الإضاءة
- `color`: الألوان الخاصة بالكائنات
